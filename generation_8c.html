Générer un fichier latex représentant le Graphe g à l'aide de TikZ.

Paramètres:
gGraphe à représenter.
destinationChemin vers le fichier de destination.
Renvoie:
0 en cas de succès, une valeur négative en cas d'erreur.

Erreurs :

  • -1 : Le fichier de destination n'a pas pu être ouvert.
  • -2 : Il y a eu une erreur lors de l'écriture.

    FILE * fichier = fopen(destination, "w");

        int erreurs = 0;

        erreurs += printDebut(fichier);

        erreurs += printFin(fichier);

        fclose(fichier);

        if (erreurs == 0)
            return 0;
        else
            return erreurs;
    }
}
