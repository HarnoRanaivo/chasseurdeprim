Générer un fichier latex représentant les Graphe g et a à l'aide de Latex et TikZ.

Paramètres:
gGraphe à représenter.
aSecond Graphe à représenter;
destinationChemin vers le fichier de destination.
Renvoie:
0 en cas de succès, une valeur négative en cas d'erreur.

g doit être renseigné, a est optionnel (utiliser NULL).

Erreurs :

  • -1 : Le fichier de destination n'a pas pu être ouvert.
  • -2 : Il y a eu une erreur lors de l'écriture.

{
    FILE * const fichier = fopen(destination, "w");

    if (fichier == NULL)
    {
        return -1;
    }
    else
    {
        Bool erreurs = FAUX;

        erreurs = printDebut(fichier) == VRAI ? VRAI : erreurs;
        erreurs = printGraphe(g, fichier) == VRAI ? VRAI : erreurs;
        if (a != NULL)
        {
            fprintf(fichier, "\n\n\\chapter{Arbre couvrant minimum}\n");
            erreurs = printGraphe(a, fichier) == VRAI ? VRAI : erreurs;
        }
        erreurs = printFin(fichier) == VRAI ? VRAI : erreurs;

        fclose(fichier);

        if (erreurs)
            return -2;
        else
            return 0;
    }
}
