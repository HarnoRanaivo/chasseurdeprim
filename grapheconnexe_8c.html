Ajout d'une arête dans un graphe connexe.

Paramètres:
gGrapheConnexe où ajouter l'arête.
aÉxtrémité de l'arête.
bÉxtrémité de l'arête.
pPoids de l'arête.
Précondition:
gEstVide(g) || gEstConnexe(g)
Renvoie:
GrapheConnexe.

N'ajoute l'arête entre les Sommet a et b que si le GrapheConnexe g reste connexe apprès l'ajout de l'arête :

  • Si aucun des deux Sommet n'existe, g reste intact.
  • Si l'un des deux Sommet existe, on peut ajouter l'autre Sommet au GrapheConnexe avant d'ajouter l'arête.
  • Si les deux Sommet existent, on peut ajouter l'arête.

Définition à la ligne 26 du fichier grapheconnexe.c.

{
    if (gEstVide(g) || gExisteSommet(g, a) || gExisteSommet(g, b))
        g = gAjouterArete(g, a, b, p);

    return g;
}

aExtrémité de l'arête.
bAutre extrémité de l'arête.
Précondition:
gEstConnexe(g)
Renvoie:
GrapheConnexe.

Supprime l'arête entre les Sommet a et b, si et seulement si le GrapheConnexe g reste connexe.

Définition à la ligne 52 du fichier grapheconnexe.c.

{
    if (gExisteArete(g, a, b))
    {
        Ent p = gPoidsArete(g, a, b);
        g = gSupprimerArete(g, a, b);

        if (!gEstConnexe(g))
            return gAjouterArete(g, a, b, p);
        else
            return g;
    }
    else
        return g;
}
